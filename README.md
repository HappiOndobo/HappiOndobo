## 🚀 Featured Projects

### 🛡️ RBSF — Risk-Based Security Framework for IoT Devices
*Master's Thesis Project*

An ML-driven framework that scores IoT network traffic by risk level (Low / Medium / High / Critical) rather than just flagging attack vs. normal — combining a supervised Random Forest classifier, an unsupervised Isolation Forest anomaly detector, and a weighted risk-fusion formula **R = 0.55×C + 0.30×A + 0.15×H**.

| Phase | Dataset | Accuracy | AUC-ROC | Status |
|-------|---------|----------|---------|--------|
| Phase 1 | Synthetic (8,900 records) | 98.37% | 0.9996 | ✅ Complete |
| Phase 2 | TON_IoT real (461,043 records) | 97.99% | 0.9988 | ✅ Complete |

`Python` `scikit-learn` `imbalanced-learn` `pandas` `matplotlib`

---

### 🛰️ Network Traffic Anomaly Visualizer
*Portfolio Project*

Interactive dashboard for unsupervised anomaly detection on any network traffic CSV. Upload your data, tune the sensitivity slider, and get a scored, downloadable report — no labeled attack data needed.

- Isolation Forest engine with normalized 0–1 risk scoring
- Interactive Plotly charts: score distribution, risk breakdown, timeline
- Works with TON_IoT, UNSW-NB15, CIC-IDS exports, or any network flow CSV

`Python` `Streamlit` `Plotly` `scikit-learn`
